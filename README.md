## Description
Energy policies have wide-ranging implications for the entire energy-economic system. Nexus-e, a highly integrated energy systems modelling platform, addresses the interaction between the different components of the system by linking interdisciplinary modules. It cuts across the boundaries of established fields of knowledge by combining existing and new tools from different disciplines.

Nexus-e originally consists of the following five interlinked modules.

- General Equilibrium Module for Electricity (GemEl)
- Centralized Investments Module (CentIv)
- Distributed Investments Module (DistIv)
- Electricity Market Module (eMark)
- Network Security and Expansion Module (Cascades)

This open-source version only contains the CentIv module for now.

More information on the Nexus-e platform can be found on our website <https://nexus-e.org/>.
