## SOLID principles
The SOLID principles are five important design principles of Object Oriented Programming that, we think, are good to keep in mind.

The [SOLID principles wikipedia page](https://en.wikipedia.org/wiki/SOLID) about these principles gives a short description.

This 20 minutes long [video](https://www.youtube.com/watch?v=pTB30aXS77U) show the application of these principles in a Python example (it's great, watch it).
