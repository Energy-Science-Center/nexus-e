Scripts in this folder are to post-process the Nexus-e results for the webviewer. Information about the post-process can be found at the [Nexus-e documentation](https://nexus-e.readthedocs.io/en/latest/visualization_tool.html#postprocess).

Instructions on how to use the stand-along script `postProcess_batch.m` can be found in the script. Customizations can be made within the "To be edited" section in the script.
